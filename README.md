# Student Registration Form

## About The Project

This project is a student registration form created using HTML.

The purpose of this assignment was to practice HTML forms, input fields, tables, radio buttons, checkboxes, and form structure to collect student information.

## Technologies Used

- HTML5

## Features

- Student information form
- Text input fields
- Date picker
- Email and phone input
- Gender selection
- Address textarea
- Hobby selection using checkboxes
- Qualification table
- Course selection options
- Submit and reset buttons

## What I Learned

Through this assignment, I learned:

- How to create HTML forms
- How to use different input types
- How radio buttons and checkboxes work
- How to create nested tables
- How to organize form data properly
- How to use labels with form elements
